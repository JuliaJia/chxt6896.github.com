h2. 必要条件

* 注册<a href="https://github.com/"> Github </a>，且已经上手

* 热爱开源，想要贡献自己博文的力量

h2. 致合作者

既然是合作，那就是要以合作者 Collaborators 的身份加入，首先要先向第一写作者（就是我啦）提交申请，发送邮件到 <b>chxt6896@gmail.com</b>，用你的真诚打动我，附上你的 Github 用户名。

具体的方式请详细阅读<a href="http://www.ossxp.com/doc/gotgithub/04-work-with-others/020-shared-repo.html">这篇文章</a>，采用的是变基推送。
