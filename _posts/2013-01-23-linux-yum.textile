yum 是一个用于管理 rpm 包的后台程序，用 python 写成，可以非常方便的解决 rpm 的依赖关系。在建立好 yum 服务器后，yum 客户端可以通过 http、ftp 方式获得软件包，并使用方便的命令直接管理、更新所有的 rpm 包，甚至包括 kernel 的更新。现在把常用的 yum 命令参数整理如下:

h2. 列举包文件

列出资源库中所有可以安装或更新的 rpm 包

{% highlight bash linenos %}
$ sudo yum list
{% endhighlight %}

列出资源库中特定的可以安装或更新以及已经安装的 rpm 包

{% highlight bash linenos %}
# 列出名为 perl 的包
$ sudo yum list perl
# 列出 perl 开头的包
$ sudo yum list perl*
{% endhighlight %}

列出资源库中所有可以更新的 rpm 包

{% highlight bash linenos %}
$ sudo yum list updates
{% endhighlight %}

列出已经安装的所有的 rpm 包

{% highlight bash linenos %}
$ sudo yum list installed
{% endhighlight %}

列出已经安装的但是不包含在资源库中的 rpm 包

{% highlight bash linenos %}
$ sudo yum list extras
{% endhighlight %}

注: extras 是 repos.d 中定义的资源列表名称

h2. 列举资源信息

列出资源库中所有可以安装或更新的 rpm 包的信息

{% highlight bash linenos %}
$ sudo yum info
{% endhighlight %}

列出资源库中特定的可以安装或更新以及已经安装的 rpm 包的信息

{% highlight bash linenos %}
# 列出 perl 包信息
$ sudo yum info perl
# 列出 perl 开头的所有包的信息
$ sudo yum info perl*
{% endhighlight %}

列出资源库中所有可以更新的 rpm 包的信息

{% highlight bash linenos %}
$ sudo yum info updates
{% endhighlight %}

列出已经安装的所有的 rpm 包的信息

{% highlight bash linenos %}
$ sudo yum info installed
{% endhighlight %}

列出已经安装的但是不包含在资源库中的 rpm 包的信息

{% highlight bash linenos %}
$ sudo yum info extras
{% endhighlight %}

h2. 搜索

搜索匹配特定字符的 rpm 包

{% highlight bash linenos %}
# 在包名称、包描述等中搜索
$ sudo yum search perl
{% endhighlight %}

搜索有包含特定文件名的 rpm 包

{% highlight bash linenos %}
$ sudo yum provides realplay
{% endhighlight %}

h2. 管理包

安装 rpm 包

{% highlight bash linenos %}
# 安装 perl 包
$ sudo yum install perl
# 安装perl 开头的包
$ sudo yum install perl*
{% endhighlight %}

删除 rpm 包,包括与该包有倚赖性的包

{% highlight bash linenos %}
# 会删除 perl-* 所有包
$ sudo yum remove perl*
{% endhighlight %}

软件组管理

{% highlight bash linenos %}
# 安装指定的组
$ sudo yum groupinstall "Chinese Support"
# 安装了的组成员软件包更新
$ sudo yum groupupdate "Chinese Support"
# 安装了的组和可以安装的组一览显示
$ sudo yum grouplist "Chinese Support"
# 删除指定的组
$ sudo yum groupremove "Chinese Support"
# 指定组所包含的软件包显示
$ sudo yum groupinfo "Chinese Support"
{% endhighlight %}

h2. 更新

检查可更新的 rpm 包

{% highlight bash linenos %}
$ sudo yum check-update
{% endhighlight %}

更新所有的 rpm 包

{% highlight bash linenos %}
$ sudo yum update
{% endhighlight %}

更新指定的 rpm 包,如更新 kernel 和 kernel source

{% highlight bash linenos %}
$ sudo yum update kernel kernel-source
{% endhighlight %}

大规模的版本升级,与 yum update 不同的是,连旧的淘汰的包也升级

{% highlight bash linenos %}
$ sudo yum upgrade
{% endhighlight %}

h2. 清空缓存

清除暂存中 rpm 包文件

{% highlight bash linenos %}
$ sudo yum clean packages
{% endhighlight %}

清除暂存中 rpm 头文件

{% highlight bash linenos %}
$ sudo yum clearn headers
{% endhighlight %}

清除暂存中旧的 rpm 头文件

{% highlight bash linenos %}
$ sudo yum clean oldheaders
{% endhighlight %}

清除暂存中旧的 rpm 头文件和包文件

{% highlight console linenos %}
$ sudo yum clearn all
# yum
{% endhighlight %}
